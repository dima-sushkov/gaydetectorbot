01 / Введение

Наш подход к данным

GayDetector Bot («Бот») собирает минимально необходимый объём данных для обеспечения игровой механики. Мы не занимаемся монетизацией пользовательских данных, не передаём их третьим лицам и не используем в каких-либо целях, кроме функционирования Бота.

✓ Мы никогда не продаём ваши данные. Мы никогда не передаём их рекламодателям. Ваши данные используются исключительно для работы игровой механики.

02 / Собираемые данные

Какие данные мы храним

При регистрации в игре (команда !пидордня) Бот сохраняет следующие данные:

Тип данных Назначение Хранение
Discord User ID Идентификация участника, упоминание в результатах До удаления участника
Discord Guild ID Разделение статистики между серверами До удаления участника
Отображаемое имя (nickname) Отображение в таблице статистики До удаления участника
Счётчик побед Ведение статистики игр До сброса или удаления
Временные метки игр Ограничение одной игры в 24 часа История игр сервера

Бот не собирает и не хранит: содержание переписки, голосовые данные, email-адреса, IP-адреса, информацию об устройствах или любые другие персональные данные.

03 / Использование данных

Как используются данные

Собранные данные используются исключительно для:

  • Случайного выбора участника при запуске игровой рулетки
  • Ведения таблицы статистики в рамках одного сервера
  • Предотвращения повторного запуска игры в течение 24 часов
  • Отображения результатов и упоминания победителя через @mention
04 / Ваши права

Управление своими данными

Каждый пользователь имеет возможность управлять своим участием в игре:

  • Удаление данных: Администратор сервера может удалить вас из игры командой !исключить @вы — все ваши данные будут немедленно удалены из базы данных
  • Сброс статистики: Администратор сервера может полностью обнулить все данные командой !сброспидоров
  • Добровольность: Участие в игре начинается только после явной регистрации командой !пидордня

Если вы хотите удалить свои данные, но не являетесь администратором сервера — обратитесь к администратору того сервера Discord, на котором используется Бот.

05 / Хранение и безопасность

Где и как хранятся данные

Данные хранятся в локальной базе данных SQLite на сервере хостинг-платформы Railway. Доступ к базе данных имеет только владелец развёртывания Бота.

Мы принимаем разумные меры для защиты данных от несанкционированного доступа, однако не можем гарантировать абсолютную безопасность любой системы передачи данных через интернет.

06 / Дети

Возрастные ограничения

Бот предназначен для пользователей, соответствующих минимальным возрастным требованиям платформы Discord (13 лет, в некоторых юрисдикциях — 16 лет). Мы не собираем намеренно данные лиц, не достигших допустимого возраста. Если вам стало известно об использовании Бота несовершеннолетним — пожалуйста, сообщите администратору соответствующего сервера.

07 / Изменения политики

Обновление документа

Мы можем обновлять настоящую Политику конфиденциальности. Актуальная версия всегда доступна по данному адресу с указанием даты последнего обновления. Продолжение использования Бота после изменений означает согласие с обновлённой Политикой.
