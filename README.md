# 🎲 GayDetector Bot

Развлекательный Discord-бот для игры «Пидор дня». Каждый день случайно выбирает одного из зарегистрированных участников, ведёт статистику и в конце года объявляет главного пидора года.

Данные хранятся в PostgreSQL и не сбрасываются.

## Команды

### 🎮 Игровые команды

| Команда | Описание |
|---|---|
| `!пидордня` | Зарегистрироваться участником |
| `!ктопидор` | Запустить рулетку (1 раз в сутки) |
| `!топпидоров` | Топ-10 статистика за всё время |
| `!пидоргода` | Объявить пидора года (только 31 декабря) |
| `!пидоргода2026` | Посмотреть архив пидора конкретного года |
| `!пидорхелп` | Показать список команд |

### 🛠️ Команды для администраторов

| Команда | Описание |
|---|---|
| `!исключить @user` | Исключить участника из рулетки (статистика сохраняется) |
| `!сброспидоров` | Обнулить статистику вручную (требует подтверждения) |
| `!setканал` | Установить текущий канал для авторулетки |
| `!setвремя 09:00` | Установить время авторулетки (UTC) |
| `!delканал` | Отключить авторулетку |
| `!resetканал` | Сбросить и переустановить настройки авторулетки |
| `!autoинфо` | Показать текущие настройки авторулетки |

## Логика игры

- Участие добровольное — игрок сам регистрируется командой `!пидордня`
- Рулетка запускается не чаще 1 раза в 24 часа
- Исключённый игрок выпадает из рулетки, но его статистика сохраняется
- Если игрок был исключён — он может снова зарегистрироваться командой `!пидордня`

---

## Логика пидора года

- Команда `!пидоргода` доступна только **с 31 декабря**
- После объявления победителя статистика сбрасывается автоматически, победитель записывается в архив
- До следующего 31 декабря команда заблокирована
- Архив прошлых лет доступен через `!пидоргода2025`, `!пидоргода2026` и т.д.
- После сброса все исключённые игроки автоматически возвращаются в игру

---

## Авторулетка

- Каждый сервер настраивает свой канал и время независимо
- Время указывается в **UTC** — не забудь пересчитать под свой часовой пояс
- Киев / Москва летом (UTC+3): `!setвремя 09:00` = 12:00 по местному времени
- Если канал не настроен — авторулетка не запускается
- Авторулетка не запустится если рулетка уже была сыграна вручную в этот день

---

## Данные и приватность

- Бот хранит только Discord User ID, никнейм и счётчик побед
- Данные каждого сервера полностью изолированы от других серверов
- База данных PostgreSQL — данные не теряются при обновлениях бота
