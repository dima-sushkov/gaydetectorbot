# 🎲 GayDetector Bot

Развлекательный Discord-бот для игры «Пидор дня». Каждый день случайно выбирает одного из зарегистрированных участников, ведёт статистику, комментирует стрики и в конце года объявляет главного пидора года. Поддерживает AI-агента на базе Groq.

## Команды

### 🎮 Игровые команды

| Команда | Описание |
|---|---|
| `!пидордня` | Зарегистрироваться участником |
| `!ктопидор` | Запустить пробивку (1 раз в сутки) |
| `!топпидоров` | Топ-10 статистика за всё время |
| `!пидоргода` | Объявить пидора года (только 31 декабря) |
| `!пидоргода2026` | Посмотреть архив пидора конкретного года |
| `!пидорхелп` | Показать список команд |

### 🛠️ Команды для администраторов

| Команда | Описание |
|---|---|
| `!исключить @user` | Исключить участника из пробивки (статистика сохраняется) |
| `!сброспидоров` | Обнулить статистику вручную (требует подтверждения) |
| `!setканал` | Установить текущий канал для автопробивки |
| `!delканал` | Отключить автопробивку |
| `!resetканал` | Сбросить и переустановить настройки автопробивки |
| `!autoинфо` | Показать текущие настройки автопробивки |

## Логика игры

- Участие добровольное — игрок сам регистрируется командой `!пидордня`
- Пробивка запускается не чаще 1 раза в 24 часа
- Исключённый игрок выпадает из пробивки, но его статистика сохраняется и учитывается при подведении итогов года
- Если игрок был исключён — он может снова зарегистрироваться командой `!пидордня`

---

## Стрики

- Если один и тот же участник побеждает **2 дня подряд** — бот добавляет дерзкий комментарий после объявления победителя
- Если участник побеждает **3 и более дней подряд** — бот тегает всех участников сервера с особым объявлением

---

## Автопробивка

- Каждый день в **23:59 UTC** бот проверяет — если никто не запустил пробивку вручную, запускает её автоматически
- Каждый сервер настраивает свой канал командой `!setканал`
- Если канал не настроен — автопробивка не запускается

---

## Логика пидора года

- Команда `!пидоргода` доступна только **с 31 декабря**
- **31 декабря в 23:59 UTC** бот запускает годовую пробивку автоматически
- После объявления победителя статистика сбрасывается, победитель записывается в архив
- До следующего 31 декабря команда заблокирована
- Архив прошлых лет: `!пидоргода2025`, `!пидоргода2026` и т.д.
- После сброса все исключённые игроки автоматически возвращаются в игру

---

## Итоги недели

- Каждое **воскресенье в 20:00 по Киеву** (17:00 UTC) бот публикует итоги недели в канале автопробивки
- AI генерирует смешной репортаж в стиле комик-шоу с именами победителей, стриками и теми кто не попался

---

## AI Агент

- Тегни бота в чате — он ответит в дерзком стиле комьюнити через GPT-4o-mini
- Бот понимает намерения и может выполнять команды голосом:
  - *"запусти пробивку"* → запускает `!ктопидор`
  - *"покажи статистику"* → показывает топ
  - *"запиши меня"* → регистрирует в игре
- Помнит контекст диалога (последние 10 сообщений)
- Перед каждым ответом показывает анимацию "печатает..." с задержкой

## Данные и приватность

- Бот хранит только Discord User ID, никнейм и счётчик побед
- Данные каждого сервера полностью изолированы от других серверов
- База данных PostgreSQL — данные не теряются при обновлениях бота
